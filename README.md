# INSTALLATION & USAGE

	
      $ git clone https://github.com/mkdirlove/GHIBLI-WEBSTORE.git
      $ cd GHIBLI-WEBSTORE
      $ python3 -m pip install -r requirements.txt
      $ python3 server.py
      
      Open your web browser and navigate to: http://127.0.0.1:5000
